01 Overview

Disentangled Representation Learning for Controllable Generation (DRL for Real)

Disentangled Representation Learning (DRL) shows promise for enhancing AI's fundamental understanding of the world, potentially addressing hallucination issues in language models and improving controllability in generative systems. Despite significant academic interest, DRL research remains confined to synthetic scenarios due to a lack of realistic benchmarks and unified evaluation metrics.

ICCV 2025 DRL for Real Workshop aims to bridge this gap by introducing novel, realistic datasets and comprehensive benchmarks for evaluating DRL methods in practical applications. We will focus on key areas including controllable generation and autonomous driving, exploring how DRL can advance model robustness, interpretability, and generalization capabilities.

02 Dataset

DRL for Real Dataset

The DRL for Real dataset is specifically designed for disentangled representation learning research, covering multiple realistic scenarios and diverse disentanglement factors. It consists of two main categories aligned with our competition tracks:

  • General Image Subset: Contains approximately 25,000 images across several distinct categories, such as human faces, animals, plants, etc. Images in this subset are partially synthesized using text-to-image generative models, while others are derived from controllable generation video sources such as time-lapse photography. This dataset features simultaneous variations in multiple factors, challenging models to disentangle complex, interrelated attributes.

  • Autonomous Driving Subset: A vehicle-centered dataset featuring seven camera perspectives:
    • Six outward-facing cameras positioned around the vehicle
    • One top-down overview camera

    Each sample consists of 50-100 frames extracted from videos captured by these seven cameras, with simultaneous variations in vehicle motion state, weather conditions (rainy, foggy, sunny), lighting, and other environmental factors. The dataset includes 12 distinct vehicle types and more than 5 background scenarios, systematically generated using the CARLA simulator.

03 Challenge

Our competition is designed to advance the field of disentangled representation learning through realistic benchmarks and comprehensive evaluation metrics. The challenge is divided into two main tracks:

Competition Tracks

1. Single-Factor Track

This track provides over 10,000 images covering multiple scenarios, dozens of objects, and hundreds of single-factor variations with text descriptions. Participants must use disentanglement models for this track.

Evaluation Metrics
  • Image Quality (50%): Measured using Fréchet Inception Distance (FID)
  • Latent Variable Disentanglement (50%): Evaluated using Bidirectional DCI, which extends the traditional DCI metric by also measuring how perturbations in real images affect the corresponding latent variables in the encoder

2. Multi-Factor Track

This track is divided into two sub-categories, each with two separate leaderboards:

2.1 General Image Dataset
  • Overall Leaderboard: Open to all methods. Evaluation uses LLM+VQA to assess:
    • Correct attribute generation based on given text
    • Change intensity scoring across a set of images
    • Ratio of intended attribute changes to unintended changes (closer to 1 is better)
  • Disentanglement Leaderboard: Only for disentanglement models, using the same metrics as the Single-Factor Track
2.2 Autonomous Dataset

A vehicle-centered dataset featuring seven camera perspectives:

  • Six outward-facing cameras positioned around the vehicle
  • One top-down overview camera

Each sample consists of 50-100 frames with simultaneous variations in vehicle motion state, weather conditions, lighting, and other environmental factors.

  • Overall Leaderboard: Evaluation uses Structural Similarity Index Measure (SSIM) to compare reconstructed frames with ground truth
  • Disentanglement Leaderboard: Only for disentanglement models, using the same metrics as the Single-Factor Track
